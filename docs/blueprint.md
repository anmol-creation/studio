# **App Name**: Ani AI

## Core Features:

- Text Chat: Real-time text chat with support for multiple languages (Hindi, English, Hinglish) and auto language detection and manual language switching.
- Voice Chat: Live voice chat with speech-to-text and text-to-speech conversion, supporting the same languages as text chat.
- Memory Engine: Project-based memory system to store and manage conversations, with owner approval required before saving any data.
- AI Powered Assistant: Acts as a private AI assistant using stored memory, text chat, and voice chat, using reasoning as a tool to remember when owner-approved memories should be incorporated into responses.
- Internet Fetch: Detects when a query requires internet data, performs web searches, extracts relevant information from trusted pages, and presents the results to the AI, which decides when or whether to present results to the user. Results are stored only if the owner approves. Uses reasoning as a tool.
- Admin Control Panel: Allows the user to view, edit, delete, and lock memory entries; manage projects, phases, and levels; and control what the AI is allowed to remember.

## Style Guidelines:

- Primary color: Gentle blue (#64B5F6) for a calming and trustworthy feel.
- Background color: Light gray (#F5F5F5) for a clean and unobtrusive backdrop.
- Accent color: Soft orange (#FFCC80) to highlight important actions and information.
- Body and headline font: 'PT Sans', sans-serif, combines a modern look and a little warmth or personality.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, modern icons to represent different actions and categories within the app.
- Subtle, smooth animations for transitions and user interactions to provide a polished and intuitive experience.
